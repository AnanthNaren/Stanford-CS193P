cardsInvolved

Stores the current state of the card matching game. It has a NSNumber object at the 0th index, which represents the current matching state, NotCheckedYet -0 , MATCH_SUCCESS -1, MATCH_FAILED -2 which is then followed by array of cards involved on that match.
